This is a Kaggle competition project which predicts the tag of stack overflow questions.
The main idea behind the poject is to build the model with the help of Classification technique . The classication technique i have used to buil the model is the Naive bayes classifier. Once the model is complete the test data is passed to predict the tags for those questions.

The test data was provided by Stackoverflow.com
The main file to run the poject is Datamine.java
